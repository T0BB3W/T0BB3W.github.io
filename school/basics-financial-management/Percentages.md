## Slide 3: The base value

Base value is the value from which the percentage is calculated from

The base value is always 100 %

How to recognize a base value?
 the value that comes after the word ”of” or ”out of”
 in changes, the old value
 in comparison, the value after the word ”than”
 the value that describes the whole
 In some cases the base value is determined, e.g. in VAT (value added tax)
