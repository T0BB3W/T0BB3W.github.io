# What are the differences in between Windows 11 and Linux Ubuntu?

Windows 11 is a proprietary operative system, which means that users must purchase licenses to use it legally, however nowadays the keys are generally in the hardware, at least in PC’s which means if you were to install windows from a flash drive or similar, the license would automatically be read and used by the OS.

Ubuntu is free and open-source, developed by Canonical.

Windows 11 features a polished and modern GUI with a load of features (and bloatware ☹)

Ubuntu’s Interface is very customizable and minimalistic.

Windows 11 supports a wide range of commercial software and comes pre installed with many protocols (think directX etc.), also in W11 the installing and updating of drivers is automatic, which is a welcome change from earlier windows versions.

Ubuntu’s compatability is limited compared to windows, some commercial software is available via compatability layers or emulators.

Windows 11 Requires modern hardware for example 4gb ram and 64gb storage at minimum, where as Ubuntu is very lightweight and only uses a fraction of the compute required for windows.

To summarize Windows 11 is the choice for consumers, businesses and gamers who need an user friendly interface with broad software support, where as Ubuntu is more popular among technically skilled audiences such as: developers, server administrators, and enthusiasts that value open-source principles, customization and controls. For security, Windows 11 relies on windows defender heavily, Ubuntu uses permissions based system, and there are generally fewer viruses targeting linux. There is a lot more technical differences which I do not feel the need to go into depth on such as: File system, Command line and customization.

# What are directories?

A directory is a logical container used to organize and store files and other subdirectories within a file system, providing a hierarchical structure for file management, giving users the ability to group related files together. In Windows 11 dir’s are commonly referred to as folders and can be accessed easily through Windows File Explorer. In ubuntu and other linux systems, directories are managed through the terminal or other file managers. Directories contain metadata such as permissions, file names and timestamps.
