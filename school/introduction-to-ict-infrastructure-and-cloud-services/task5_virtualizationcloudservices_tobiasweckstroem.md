# What benefits does Virtualization offer?

Virtualization allows for multiple operating systems or instances to run on the same physical computer by creating Virtual Machines or containers, basically abstractifying the physical limitations of the computer to allow more efficient allocation of it’s available resources. This leads to reduced hardware costs, enhances scalability, simplifies software testing, and allows flexible environments for running many operative systems or apps on a single machine. For example, if you were experimenting with kernel level permissions or network penetration testing, it would be advantageous to do this sort of testing on a virtual machine running on your personal computer, to avoid risks of bricking your computer, or legal ramifications regarding the penetration testing.
